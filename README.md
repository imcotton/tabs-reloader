Tabs Reloader
=============

Reload tabs in current window

* single-click to reload pinned tabs
* double-click to reload normal tabs


Installation
------------

#### Google Chrome

* [Chrome Web Store][cws] **(recommended)**  
* [web page][wp] _(if you're having trouble accessing above)_
